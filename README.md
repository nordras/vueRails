# Prototype based VueOnRails SSR

#### Project made for study issues and prototype some mechanics
```
Install before postgresql
git clone origin git@github.com:nordras/vueRails.git
bundle install
rails db:create
rails db:migrate
yarn
rails vue:setup
gem install foreman
foreman start
Play fun :)
```
